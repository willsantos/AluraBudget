## Descrição do Projeto

Alura Budget foi desenvolvido durante o Alura Challenge Back-End 2ª Edição.

Para esse desafio optei por utilizar o C# com o .Net, ainda não tinha feito nada
completo com essa stack, e foi interessante revisitar o .Net já que a ultima 
vez que usei foi na versão 1.1 ainda com o vb.


## Desafio
O Alura Challenge Back-End não é um curso e sim um desafio para alunos, onde são apresentados cards no Trello com as especificações de cada sprint.

### Sprint 1
Api REST com CRUD para despesas e receitas

### Sprint 2
Solicitaram novas funcionalidades:

* Busca de receitas e despesas por descrição
* Listagem de receitas e despesas de um determinado mês
* Resumo de determinado mês (total de receitas, total de despesas, saldo final e valor total agrupado por categoria)

### Sprint 3
implementar um mecanismo de autenticação na API, para que apenas usuários autenticados possam interagir com ela.

Além disso, também será necessário realizar o deploy da API em algum provedor Cloud
